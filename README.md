# To-Do List 📝

A simple and easy-to-use To-Do List web app to manage your daily tasks.

## ✨ Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Stores tasks in your browser (local storage)

## 🚀 Technologies Used

- HTML
- CSS
- JavaScript
